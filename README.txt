This is the home for xcsde-rngval XML Validator.

This is a Validator for XML Documents using RELAX NG schemas, supporting
XML Catalogs, Schematron, XML Entities.  Some specific use cases known to
work are XML Includes nesting and DocBook 5.0 source files.

You run this validator with a command like

    java -switches...  schema.rng file.xml...
